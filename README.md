# AI Productivity Platform - Backend

## 🚀 Overview

Production-ready backend server for the AI Productivity Platform. Built with Node.js, Express, and MongoDB, integrated with Google's Gemini AI for advanced AI capabilities.

## 📋 Features

- **Complete Authentication System**: JWT-based auth with email verification
- **AI Chatbot**: Real-time AI conversations with Gemini Pro
- **Image Analysis**: Computer vision using Gemini Vision API
- **Code Generation & Debugging**: Multi-language code assistance
- **Document Summarization**: PDF and text summarization
- **Multimodal Processing**: Combined text, image, and document analysis
- **User Dashboard**: Analytics and activity tracking
- **Admin Panel**: User management and platform statistics
- **Security**: Rate limiting, Helmet, input validation, XSS protection

## 🛠️ Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: JWT & bcrypt
- **AI**: Google Gemini API (Pro & Vision)
- **File Upload**: Multer
- **Security**: Helmet, CORS, Rate Limiting
- **Validation**: express-validator
- **Logging**: Morgan
